# Websphere Solution — Company Website

Static multi-page website for **Websphere Solution Pvt. Ltd.**, a digital marketing company based in Delhi NCR, India. The site covers the company's services, packages, courses, careers, team, and contact information.

## Tech Stack

- HTML5, CSS3, JavaScript
- Bootstrap 5 (customized)
- SCSS (Bootstrap source included)
- Owl Carousel — image/content sliders
- WOW.js + Animate.css — scroll-triggered animations
- CounterUp.js — animated counters
- jQuery Easing — smooth animations
- Font Awesome 5 + Bootstrap Icons
- Google Fonts — Poppins, Roboto
